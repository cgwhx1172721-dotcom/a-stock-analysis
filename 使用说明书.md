> 网址：**https://a-stock-analysis.vercel.app**
>
> 本工具仅供参考，不构成投资建议。股市有风险，投资需谨慎。

---

## 一、产品简介

A 股分析助手是一款专为 A 股投资者设计的股票分析工具，只需输入 6 位股票代码，即可一键查看：

| 分析模块 | 说明 |
|---------|------|
| 主力资金 | 近 3 天主力净流入/流出情况 |
| 量能分析 | 量比和换手率，判断市场活跃度 |
| 赛道热度 | 是否属于当前市场热门板块 |
| 投资逻辑 | 综合各维度信号给出买入逻辑评估 |
| 基本面分析 | 市值、市盈率、市净率评估 |
| 价格位置 | 当前价格处于近 90 天的哪个位置 |
| 技术指标 | 均线系统、RSI、MACD |
| 风险分析 | 超买风险、主力撤退风险等 |
| 综合建议 | 综合所有信号给出操作建议 |
| 我的持仓 | 输入持股数量和成本，计算实时盈亏 |
| 止盈止损 | 三档止盈价 + 参考止损价 + 风险回报比 |

## 二、电脑端使用方法

### 第一步：打开浏览器

推荐使用 **Chrome**、**Edge**（Windows 自带）或 **Safari**（苹果 Mac 自带）。

### 第二步：输入网址

在浏览器地址栏输入，按回车打开：

```
a-stock-analysis.vercel.app
```

> ⚠️ **注意**：该网址在中国大陆可能需要 VPN 才能访问，请先确保网络正常。

### 第三步：搜索股票

1. 在首页中间的输入框里，输入 **6 位股票代码**
   - 例如：贵州茅台 → `600519`，比亚迪 → `002594`，宁德时代 → `300750`
2. 点击橙色的「**开始分析**」按钮
3. 等待约 3～5 秒，分析报告自动展示

> 也可以直接点击首页下方「热门股票」中的名字快速查看

### 第四步：阅读分析报告

页面从上到下依次展示所有分析模块。信号颜色含义：

- 🟠 **橙色底色**：积极信号，值得关注
- ⚪ **白色底色**：中性信号，继续观察
- 🔴 **红色底色**：偏弱信号，注意风险

### 第五步：录入我的持仓（可选）

如果已持有该股票，录入持仓可获取个性化的盈亏分析：

1. 向下滑动页面，找到「**我的持仓**」卡片
2. 点击右上角「**+ 录入**」按钮
3. 填写两项数据：
   - **持股数量**：您持有的股数，如 `1000`
   - **持仓均价**：每股平均买入价，如 `12.50`
4. 点击橙色「**保存持仓**」按钮

系统会自动计算并显示：
- 当前持仓市值和盈亏金额（盈利显示红色，亏损显示绿色）
- 触发止损时会亏损多少元
- 到达止盈一、二、三时可以盈利多少元

> 💾 数据保存在您的浏览器中，下次打开同一只股票会自动读取，无需重复填写。

### 第六步：刷新数据

点击页面右上角「**刷新**」按钮，可获取最新行情数据。

## 三、手机安装方法（变成桌面 App）

将网页安装到手机桌面，打开体验和普通 App 完全一样：全屏显示、快速启动，方便每天查看。

---

### 苹果手机（iPhone / iPad）安装步骤

> ⚠️ **必须使用自带的 Safari 浏览器**，其他浏览器（如 Chrome）在苹果手机上不支持安装到桌面。

**步骤如下：**

1. 打开 **Safari**，在地址栏输入 `a-stock-analysis.vercel.app` 并打开
2. 等页面完全加载后，点击屏幕**底部工具栏正中间**的「**分享**」按钮
   - 图标样式：一个方框，里面有一个向上的箭头 ⬆️
3. 在弹出的菜单里向下滑动，找到「**添加到主屏幕**」，点击
4. 上方会出现名称输入框，可以改成「**A股助手**」，然后点右上角「**添加**」
5. 退出 Safari，回到手机桌面，即可看到 A 股助手的橙色图标

**日后使用：** 直接点桌面图标打开，全屏显示，就像普通 App 一样，无需打开浏览器。

---

### 安卓手机安装步骤

> 推荐使用 **Chrome 浏览器**（大多数安卓手机已默认安装，或在应用商店搜索 Chrome 下载）

**方法一：浏览器自动提示（最简单）**

1. 打开 Chrome，输入 `a-stock-analysis.vercel.app` 并打开
2. 如果 Chrome **底部出现提示横幅**：「添加 A股分析助手 到主屏幕」，直接点「**安装**」即可
3. 桌面会自动出现 App 图标

**方法二：手动从菜单添加（如未出现自动提示）**

1. 打开 Chrome，输入 `a-stock-analysis.vercel.app` 并打开
2. 点击浏览器右上角的「**三个点 ⋮**」菜单
3. 在菜单里找到：
   - 「**添加到主屏幕**」，或
   - 「**安装应用**」
   - （不同手机/Chrome 版本名称略有不同）
4. 在弹出框中点「**添加**」
5. 桌面出现 A 股助手图标

**日后使用：** 点击桌面图标，全屏启动，无需重新打开浏览器。

## 四、各模块使用说明

### 🔍 综合建议（页面中部）
三种结果：
- ✅ **可以关注**：主力资金、量能、赛道三个条件基本满足
- 🟡 **继续观察**：部分条件满足，等信号更明确再考虑
- ⏸️ **暂时观望**：条件不成熟，建议等待

### 📈 价格位置（进度条）
橙色进度条显示当前价格在近 90 天中的高低位置：
- 进度条**靠左**（0~30%）：价格处于低位区，相对安全
- 进度条**居中**（30~70%）：中间区域，不高不低
- 进度条**靠右**（70~100%）：价格偏高，追高需谨慎

### 🛡️ 止盈止损参考价位（页面底部）

| 项目 | 含义 | 操作建议 |
|------|------|---------|
| 参考止损价 | 价格跌到这里要考虑卖出 | 跌破此价，建议止损离场 |
| 止盈一（+5%） | 当前价上涨 5% | 可先卖出一半，锁定部分利润 |
| 止盈二（+10%） | 当前价上涨 10% | 继续减仓 |
| 止盈三（+18%） | 当前价上涨 18% | 可考虑全部卖出 |
| 风险回报比 | 每亏 1 元可能赚多少 | 1:1.5 以上表示性价比较好 |

### 💼 我的持仓（两项输入内容）

| 输入项 | 说明 | 示例 |
|--------|------|------|
| 持股数量 | 您持有的股数（整数） | `500` |
| 持仓均价 | 每股平均买入成本（元） | `15.80` |

填写后系统会自动展示：当前是否盈利、盈利/亏损多少元、到每个止盈/止损价位时的预计到账金额。

## 五、常见问题

**Q：网页打不开怎么办？**

A：该网址（vercel.app）在中国大陆需要 VPN 才能访问。请先开启 VPN，再重新打开。

---

**Q：数据是实时的吗？**

A：数据来自公开行情接口，存在约 15 分钟的延迟。收盘后展示当天收盘数据。

---

**Q：iPhone 上找不到「添加到主屏幕」？**

A：请确认使用的是系统自带的 **Safari** 浏览器（蓝色指南针图标），而不是 Chrome 或其他浏览器。只有 Safari 支持此功能。

---

**Q：安卓手机菜单里没有「添加到主屏幕」？**

A：请确认使用的是 **Chrome 浏览器**，并将 Chrome 更新到最新版本。部分国产手机浏览器（如 UC、华为自带浏览器）可能不支持，建议改用 Chrome。

---

**Q：我的持仓数据会泄露吗？**

A：持仓数据**只保存在您自己的手机/电脑浏览器里**，不会上传到任何服务器，请放心使用。

---

**Q：如何修改或删除已保存的持仓？**

A：进入该股票的分析页面 → 找到「我的持仓」卡片 → 点右上角「**修改**」→ 重新填写或点「**清除**」删除。

---

**Q：不同股票的持仓会混在一起吗？**

A：不会。每只股票的持仓数据是**独立保存**的，互不影响。

---

## 六、快速上手示例（以比亚迪为例）

1. 打开 `a-stock-analysis.vercel.app`
2. 输入 `002594`，点击「开始分析」
3. 查看各模块颜色：橙色越多，信号越积极
4. 向下滑到「止盈止损参考价位」，记住三个止盈价和一个止损价
5. 如果已持有比亚迪，找到「我的持仓」→ 点「+ 录入」→ 填写数量和均价 → 保存
6. 查看当前盈亏金额，以及涨到各止盈价时能赚多少
